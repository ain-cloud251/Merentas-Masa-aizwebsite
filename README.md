# aizwebsite
Portal koswer multimedia interaktif bertajuk "Merentas Masa" untuk rujukan pembelajaran digital Sejarah Melayu. Laman web ini dilengkapi dengan pelbagai modul nota interaktif serta kuiz uji minda bagi memudahkan pemahaman. Pengguna juga boleh berinteraksi dengan asisten sembang AI yang disediakan untuk mendapatkan bimbingan pembelajaran.
